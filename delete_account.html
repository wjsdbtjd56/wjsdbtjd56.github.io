계정 삭제 절차

Flutch 계정 삭제는 사용자의 개인정보 보호를 위해 신중하게 진행됩니다. 삭제 요청 시 모든 예측 정보와 개인 데이터가 영구적으로 제거됩니다.

  1. 앱 실행 후 프로필 또는 상단 메뉴(☰)를 탭합니다.
  2. 설정 탭으로 이동하여 계정 삭제 버튼을 선택합니다.
  3. 삭제 확인 팝업에서 “예, 삭제합니다”를 눌러 요청을 전송합니다.
  4. 삭제 요청 후 즉시 모든 데이터가 완전 파기됩니다.

삭제되는 데이터

  • 회원님의 축구 경기 예측 기록
  • 닉네임, 이메일, 로그인 토큰 등 개인 프로필 정보
  • 앱 내 저장된 기타 모든 사용자 지정 데이터

데이터 보관 정책

삭제 요청 즉시 모든 데이터가 완전 파기됩니다. 삭제 전까지는 주민등록번호, 계좌번호 등 일부 민감한 개인 정보는 AES-256 암호화 기준으로 안전하게 보관됩니다.
